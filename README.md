# GitHub Integration Dashboard

An Angular **19.x** application for connecting to GitHub via **OAuth2** and visualizing data such as organizations, repositories, commits, pull requests, issues, and users.  
Built with **Angular Material** and **AG Grid** for a modern, responsive UI.

---

## 🚀 Features

- 🔐 **OAuth2 GitHub Integration**  
  Connect and authenticate with GitHub securely.

- 🏢 **Organizations & Repositories**  
  Fetch and browse GitHub organizations and their repositories.

- 📝 **Commits, Pull Requests, Issues, Users**  
  Drill down into repository activity and contributors.

- 🎨 **Modern UI**  
  Built with Angular Material + AG Grid (pagination, filtering, sorting).

- ⚡ **Backend Integration**  
  Works with a Node.js/Express + MongoDB backend for storing integration details.

---

## 📦 Tech Stack

- **Frontend:** Angular 19, Angular Material, AG Grid  
- **Backend:** Node.js, Express, MongoDB (integration storage)  
- **Auth:** GitHub OAuth2  
- **Other:** RxJS, Zone.js, TypeScript
